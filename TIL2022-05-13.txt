- 다양한 javascript 알고리즘 (문법) 배우기 시작
number() 숫자형으로 타입을 바꿀때 - 소수점까지

parseint() 숫자형으로 타입을 바꿀때 - 정수 반환

let a = '10.223456';

let b = parseInt(a); // 10 
let c = Number(a); // 10.223456

String.trim() 대상 문자열 앞뒤에 공백문자 제거후 문자열 반환

string.split() 각 문자를 모두 분리하여 배열로 반환
string.split( separator, limit ) 분할기준(separator)과 최대 분할 개수 지정

data.split() 대상 문자열 전체를 단일 요소로 하는 배열로 반환

reduce는 total의 값과 val값이 서로 더해지면서 쌓이는 형태
