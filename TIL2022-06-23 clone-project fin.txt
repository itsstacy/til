조금 아쉬운 점이 꽤 많았다. 프론트 팀원이 갑자기 아프다고 해서 몇가지 기능을 구현을 못했다. (회사정보 디테일 페이지 등등) 그리고 실시간 채팅 부분이 로컬 환경에서 잘돌아가다가 배포하고 나서 자꾸 에러가 생겨서 결국 해결하지 못하고 회고때 멘토님께 물어봤는데 어느정도 해결을 해주셨고 좋은 조언을 많이 해주셨다.

메인페이지 -> 채용탭
- 실제 wanted 사이트를 그대로 구현
- 캐러셀 (carousels)도 사이트와 동일한 라이브러리로 구현
- 크게 두 부분: 채용공고/ 회사정보
- 채용공고와 회사정보를 두 섹션에 나눠서 뿌림
- 기업회원으로 가입하면 메인페이지 회사정보란에 노출
- 클릭하면 디테일 페이지 
 ->하단에 카카오지도로 위치 표시
 ->하단에 게시글 올린 회사정보 표시

기업회원 서비스
- 기업회원 가입시 회사 사진 여러장 추가 가능
- 기업 로그인 하면 채용공고 올리는 버튼이 navbar에 활성화됨
- 새 채용공고 올리기 기능
- 기업서비스 탭에서 자기가 올린 채용공고 관리할 수 있음
  -> 공고 수정
  -> 채용완료 하기 

개인 회원가입 및 로그인
- 구글계정으로 로그인하기 가능
